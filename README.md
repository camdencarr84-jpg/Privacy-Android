## About
*Retain* your privacy, NO ROOT. Do you act as if Google is always watching you? 
No, you probably don't. 

**BUT! Google is always listening to you.** Until I did what my scripts provide, I felt uneasy pointing a phone camera up at me, even when the Camera app wasn't running. I managed to regain my privacy, but it REALLY wasn't a user-friendly expeirence. *Yes,* even with Shizuku and Canta, it wasn't VERY user friendly. I aim to change that.
With this toolbox you can regain control of your **privacy** and mitigate, **[September](https://keepandroidopen.org)**

## How to use

Run req.py to install ADB and all other needed privileges.
Then run any needed .py files and watch it cook.



## Extras, (I hope to keep adding more.)


### Dhizuku and DEVICE OWNER!
Dhizuku is included in the package, run device_owner.py to execute that. MAKE SURE YOU ARE SIGNED OUT OF ANY GOOGLE OR SAMSUNG ACCOUNTS BEFORE RUNNING.


### EXTRA! A Switch to FOSS program!

**The included program,** switch_to_foss.py is a program that will ASSUME you have already used debloat.py and uninstalled ALL or MOST Google/Samgsung bloat. 
Run the program and ADB will auto install ALL apps (for fun, I included some games.).
